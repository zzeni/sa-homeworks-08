The square task

Excercise

With only using CSS do the following:

  1. Make he colored fields below become squares with size 200Ñ…200px
  2. When screen width is less than 400 pixels, make all the squares become as wide as the container and aligned vertically one below another (example)
  3. When the screen width is between 400 and 800 position all the squares in the four corners of the container, implementing the following rules:
    • when the screen width is between 400 and 500, make them in order: red, blue, green, black (example)
    • when the screen width is between 500 and 600, make them in order: blue, green, black, red (example)
    • when the screen width is between 600 and 700, make them in order: green, black, red, blue (example)
    • when the screen width is between 700 and 800, make them in order: black, red, blue, green (example)
  4. Finally when the width is over 800 pixels - make the squeres horizontally aligned (example)
